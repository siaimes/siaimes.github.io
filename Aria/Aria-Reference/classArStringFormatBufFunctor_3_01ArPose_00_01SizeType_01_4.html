template<typename SizeType>
class ArStringFormatBufFunctor< ArPose, SizeType >

Specialization of ArStringFormatBufFunctor for ArPose.

The format string should include three floating point values: X, Y and Theta. (e.g. "(%.2f,%.2f,%.2f)")
