Detailed Description

Contains conversion of latitude and longitude to flat map coordinates in one object with one convert() function returning a Ar3DPoint object; can internally obtain current GPS position from an ArGPS object; and can automatically use the first latitude and longitude coordinate received as the origin point of the output coordinate system.

May simplify composition with other software in some cases where you want to use single function calls and single objects rather than storing several variables (e.g. via ArFunctor object, filters, etc.), and also do not have a GPS position yet to use for the origin at time of initialization.
