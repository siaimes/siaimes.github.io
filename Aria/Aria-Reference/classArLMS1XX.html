ArLaserConnector Use ArLaserConnector to connect to a laser, determining type based on robot and program configuration parameters.

This is the ArLaser implementation for SICK LMS1xx, LMS5xx, TiM310/510 (aka TiM3xx), TiM551, TiM561, and TiM571 lasers. To use these lasers with ArLaserConnector, specify the appropriate type in program configuration (lms1xx, lms5xx, tim3xx or tim510, tim551, tim561, tim571).
