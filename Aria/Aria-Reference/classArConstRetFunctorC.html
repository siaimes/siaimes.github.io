Detailed Description

template<class Ret, class T>
class ArConstRetFunctorC< Ret, T >

Functor for a const member function with return value.

Use this when the function you are targetting is a const method. For example, use ArConstRetFunctorC<double, ArRobot>(&robot, ArRobot::getX) to create a functor targetting the ArRobot method declared with: double getX() const;.

For an overall description of functors, see ArFunctor.

Examples:
functorExample.cpp.
