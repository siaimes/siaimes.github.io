Detailed Description

This mode uses the most simple robot control commands to drive back and forth along a line repeatedly.

It can be activated by an ArNetworking "testloop" request, or by a user using custom text commands interface (if command handler is passed in constructor or addCommands() is called). Some options are added to the configuration if addToConfig() is called or an ArConfig object is given in the constructor.

Examples:
serverDemo.cpp.
