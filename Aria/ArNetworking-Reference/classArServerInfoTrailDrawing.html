Detailed Description

Stores a history of ArPose objects, and provides them as drawings to a client (such as MobileEyes) via ArServerInfoDrawings.

Positions are obtained via the supplied functor, so any object providing positions as ArPose objects may be used as the source of the trail. This functor is called when triggered by calling pull().
